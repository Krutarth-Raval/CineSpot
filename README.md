## 🌟 Key Features

✅ Explore **Trending**, **Popular**, & **Top Rated** Movies & Series  
✅ Detailed **Overview Pages** with Posters, Ratings, & Descriptions  
✅ TV Series **Seasons & Episodes** Info  
✅ 🔍 Powerful **Search** for Movies & Series  
✅ 🎨 Modern UI with **CSS Modules** & consistent design  
✅ ⚡ Super-fast performance with **Vite** build tool  
✅ 🔒 Secure API Key management with `.env`

## ⚡ Getting Started

### 🔨 Clone the Repository

```bash
git clone https://github.com/Krutarth-Raval/CineSpot.git
``` 
```bash
cd CineSpot
```
### 📦 Install Dependencies
```bash
npm install
```

### 🔑 Set up Environment Variables
```bash
VITE_TMDB_API_KEY=your_tmdb_api_key_here
```
### 🚀 Run the Application
```bash
npm run dev
```
Visit http://localhost:5173 in your browser.
