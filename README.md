🎬 CineSpot — Your Movie & Series Explorer

CineSpot is a React-based web application that allows users to explore trending, popular, and top-rated movies and TV shows. It also features a dedicated section for the latest entertainment news, offering a one-stop platform for movie lovers.

🚀 Features

Browse Trending, Popular, and Top Rated movies and series
Separate pages for Movies and TV Shows
Entertainment News section powered by News API
Clean and responsive dark-themed UI
Organized code with separate CSS files per component
Mobile-friendly experience

🔧 Tech Stack

React.js – Frontend development
TMDb API – For fetching movie and TV data
News API – For latest entertainment news and articles
React Router – For smooth navigation
Custom CSS – Component-level styling
Netlify – For deployment and hosting
Git & GitHub – For version control and collaboration

🔐 Upcoming Features

User Authentication (Login & Signup)
Watchlist and Personal Collection
Search Filters by genre, actor, and rating
UI improvements and subtle animations

CineSpot is an ongoing personal project built to learn and showcase React development, API integration, and responsive design. It’s growing with new features and enhancements regularly.
